Features

Everything you need to
automate reliably

Tapcraft gives you a visual workflow builder, durable execution, and a code-first API that AI agents can drive natively.

Visual Workflow Editor

Build directed acyclic graphs by connecting activity nodes with edges. Drag, drop, and configure — the platform generates Temporal-ready Python code from your graph automatically.

Durable Execution

Every workflow runs on Temporal. Automatic retries, configurable timeouts, and full execution history come free.

MCP Agent Integration

Connect Claude Code or any MCP-compatible agent. 30+ tools for deploying, running, and managing workflows programmatically.

Git-Backed Workspaces

Point a workspace at your git repo. Tapcraft clones it, auto-discovers activities and workflows, and keeps them in sync.

Cron Schedules

Trigger workflows on any cron schedule with timezone support. Managed through Temporal's native scheduling.

Secrets Management

Store API keys and credentials encrypted at rest with Fernet. Activities access secrets securely at runtime.

Architecture

Simple stack, serious durability

Three containers, one database, zero magic. Your agent talks MCP, the API talks Temporal, the worker executes your code.

Getting Started

Up and running in under a minute

Docker Compose handles everything. Three commands to a running platform.

1

Clone and configure

terminal
$ git clone https://github.com/tapcraft-io/tapcraft-os.git $ cd tapcraft-os $ cp .env.example .env
2

Start the stack

terminal
$ docker compose up --build ✔ temporal Started ✔ api Started → localhost:8001 ✔ worker Started
3

Launch the UI

terminal
$ cd ui && npm install && npm run dev VITE ready → http://localhost:5180
4

Connect Claude Code (optional)

~/.claude/claude_desktop_config.json
{ "mcpServers": { "tapcraft": { "command": "python", "args": ["-m", "src.mcp.server"], "cwd": "/path/to/tapcraft-os" } } }
Agent-native

Let your AI agent
run the platform

Tapcraft exposes 30+ MCP tools that let Claude Code (or any MCP-compatible agent) manage the full lifecycle without touching the UI:

  • Create and deploy workflows from code
  • Execute runs and monitor status
  • Manage secrets and schedules
  • Sync workspaces from git repos
  • Discover activities and wire them into graphs

The MCP server also exposes documentation resources at tapcraft://docs/* so agents can learn how to write activities and workflows on their own.
